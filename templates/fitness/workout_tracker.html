个性化训练建议

{% if user_info.fitness_goal == '增肌' %}
增肌训练重点
  • 增加重量训练，每个动作8-12次
  • 确保充足蛋白质摄入（每天1.6-2.2g/kg体重）
  • 大肌群训练后休息48小时
  • 保证7-9小时优质睡眠
推荐动作
  • 卧推 - 锻炼胸肌
  • 深蹲 - 锻炼下肢
  • 硬拉 - 全身复合动作
  • 引体向上 - 背部训练
{% elif user_info.fitness_goal == '减脂' %}
减脂训练重点
  • 结合有氧训练和力量训练
  • 控制热量摄入，创造热量缺口
  • 保持蛋白质摄入以维持肌肉
  • HIIT训练提高代谢
推荐动作
  • 间歇跑 - 高效燃脂
  • 循环训练 - 全身燃脂
  • 跳绳 - 方便有效的有氧
  • 波比跳 - 全身性高强度
{% else %}
新手推荐
  • 每周3-4次训练
  • 每次30-45分钟
  • 俯卧撑初学者每天10-20个
  • 注重正确姿势
中级推荐
  • 每周4-5次训练
  • 每次45-60分钟
  • 俯卧撑每天30-50个，3-4组
  • 增加负重训练
高级推荐
  • 每周5-6次训练
  • 每次60-90分钟
  • 俯卧撑每天50+个，4-5组
  • 高级变式训练
{% endif %}
